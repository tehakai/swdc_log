## Mar.15, 2019 (Fri.)  
* 稼働一周年おめでとうございます  
* 水曜日 (Mar.13) からシーズン 9 がスタート  
予選はレイク・84・ノーザンで好きなコースが揃う  
* 各予選コースにて安牌走行を探りながら走る  
* 残業続きやらプライベートのショックやらでひどく疲れた週だった  
ゆっくり余裕をもって走ったつもりだが、タイムがバラけている  

### レイク (自己ベスト: 49.591 sec.)
* シーズン 5 で詰めたときは 46.001 sec. まで迫れたので、  
今シーズンは45秒台 (45.999 sec.) を目標タイムとする
* 最初の右ヘアピンは、[100] からフルブレーキ→2速にして、  
[50]からアクセルオフでインを目指しクリップでアクセル脱出が良さそう  
* セクター2 直後の右直角は、[100]フルブレーキ→[50]ブレーキ抜いてターンイン  
クリップを抜けたらアクセルオンが良さそう
* はじめての方の参考になるよう、その後のS字と最終コーナーも書いておく  
* S字: 縁石内側の明るい緑のゾーンまで使い、なるべく直進する  
* 最終コーナー: [100] 超えたら全開で右ステア

### 84 (自己ベスト: 1'03.781)
* シーズン 6 で詰めたときは 56.600 sec. まで迫れたので、  
今シーズンは 56秒台前半 (56.499 sec.) を目標タイムとする

* セクター2 直後の左ヘアピンは、アウトから[100]フルブレーキ→[50]ブレーキ抜いてターンイン  
インに沿って走り、断続的にアクセルを入れていく  
* 左複合(コの字) は、[100] フルブレーキ→[50]ブレーキ抜いてターンイン  
* 最初の直角はクリップ通過でアクセルオン、奥の直角は左ステアを切りながらクリップを狙う  
もし曲がれなさそうだったらアクセルオフ→アクセルオンでノーズを内側に入れて脱出

### ノーザン (自己ベスト: 1'34.361)
* いままでの自己ベストが確か 1分27秒台だったので、  
今シーズンは 1分26秒台 (1'26.999) を目標タイムとする  
* 最初のヘアピンは [100]フルブレーキで3速まで落としブレーキ抜いてターンイン  
出口が見えたらアクセルオンぐらいが安牌  
* その後のS字は [100] アクセルオフ → [50] ターンイン、左の縁石踏んだらそのまま全開で右→右→左 のインをつく  
* トンネル前の左ヘアピンは [100] フルブレーキで2速まで落としブレーキ抜いてターンイン(横断歩道あたり)  
インについて出口が見えたらアクセルオン  
* トンネル後のS字は [100]ブレーキ → [50]ブレーキ抜いてターンイン  
手前の右はインをつこうとせず、奥の左のインに視線を移し、進行方向が奥の左コーナーのクリップに向いたらアクセルオン  
* その後の左ヘアピンは、[100]フルブレーキで2速まで落とし、ブレーキ抜いてターンイン  
インの縁石を踏んだらアクセルオン  
* 最終の右直角は、イン側ガードレール始まりが画面右端に到達したら右にステアを切りながらアクセルオフ  
出口が見えたらアクセルオン
